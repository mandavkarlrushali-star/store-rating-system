Store Rating System

Tech Stack

- Frontend: React.js + Bootstrap
- Backend: NestJS
- Database: MySQL
- Authentication: JWT

Features

- User Login Authentication
- Role Based Access (Admin, User, Store Owner)
- Add User
- Add Store
- View Users
- View Stores
- Search Users
- Search Stores
- Submit Store Ratings
- Admin Dashboard
- Store Owner Dashboard
- Logout Functionality

Project Structure

store-rating-frontend/ → React Frontend

StoreRatingSystem/backend/ → NestJS Backend

Run Project
Frontend
npm install
npm run dev
http://localhost:5174/
http://localhost:5173/dashboard
http://localhost:5173/view-stores
http://localhost:5173/owner-dashboard

Backend
npm install
npm run start:dev
http://localhost:3000/
http://localhost:3000/users
http://localhost:3000/ratings
http://localhost:3000/admin/dashboard
